## 功能模块

### `SendNotify.py`

通知模块，用于捕获脚本运行输出，并在脚本结束后通过青龙面板自带的 `notify.py` 推送运行结果。

如果没有找到 `notify.py`，脚本仍可运行，但不会发送通知。

青龙通知配置方式：进入青龙面板的“配置文件”页面，从44行开始，找到自己想要使用的推送方式，在双引号中填入对应的配置，脚本运行结束后会自动发送通知。

### `LoadProxy.py`

代理模块，通过配置环境变量 `proxy_server` 将脚本中的HTTP请求转发到代理服务器发起请求。

- 代理地址必须以 `http://` 或 `https://` 开头。

示例：

```bash
proxy_server=http://127.0.0.1:7890
```

## 运行方式

只需要运行 `脚本列表` 中的脚本，`SendNotify.py` 是通知模块，不需要主动运行。`LoadProxy.py` 是可选代理模块，本次新增的 Python 脚本未导入该模块。
